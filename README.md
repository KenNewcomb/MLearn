# MLearn
A collection of classic machine learning algorithms, written in Python.

## Supported Algorithms

### Supervised Learning
- Decision Trees
- Linear Regression
- Logistic Regression 
- K-Nearest Neighbors (KNN): regressor, classifier
- Multilayer Perceptron
- Support Vector Machines

### Unsupervised Learning
